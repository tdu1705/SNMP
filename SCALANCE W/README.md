## PL 
Pobieranie informacji za pomocą SNMP z punktu dostępowego i modułu klienckiego SCALANCE W.

1. Nazwa systemowa
2. Osoba kontaktowa
3. Lokalizacja systemowa
4. Numer zamówieniowy MFLB
5. Serial Number
6. Nazwa urządzenia
7. Vendor
8. Firmware Version
9. Hardware Version
10. Czas działania urządzenia
